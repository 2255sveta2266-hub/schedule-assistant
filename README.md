# Schedule Assistant — VK-бот расписания ИРНИТУ

VK-бот для получения расписания занятий Иркутского национального исследовательского технического университета (ИРНИТУ). Парсит официальный сайт, хранит данные в SQLite и отвечает через YandexGPT.

## Возможности

- Расписание любой группы (40+ групп института)
- Расписание на сегодня и завтра — одной кнопкой
- Расписание на конкретную дату в формате `01.04.2026`
- Расписание по фамилии преподавателя
- Бот запоминает группу пользователя (FSM-диалог)
- Автоматическое обновление расписания каждые 24 часа (в 03:00)
- Синхронизация расписания в Yandex AI Studio Vector Store для семантического поиска
- Fallback-режим без YandexGPT — бот работает и без ключей AI

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <url>
cd scheldule_assistant
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
venv\Scripts\activate       # Windows
source venv/bin/activate    # Linux/macOS
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Создать файл `.env`

Обязательные переменные:

```env
VK_TOKEN=ваш_токен_сообщества_вк
GROUP_ID=id_вашего_сообщества_вк
```

Опционально — для интеграции с YandexGPT:

```env
YANDEX_API_KEY=Api-Key ...
YANDEX_FOLDER_ID=b1g...
YANDEX_VECTOR_STORE_ID=...
```

Если ключи Yandex не заданы, бот работает в fallback-режиме: отвечает напрямую форматированными данными из SQLite без обращения к YandexGPT.

### 5. Запустить

```bash
python app.py
```

При первом запуске бот автоматически скачает расписание всех групп в фоне (~5–10 минут). Отвечать на сообщения он начнёт сразу.

## Команды бота

| Сообщение | Что делает |
|---|---|
| `АСУб-25-1` | Запомнить группу и показать её расписание |
| `сегодня` | Расписание на сегодня (для запомненной группы) |
| `завтра` | Расписание на завтра (для запомненной группы) |
| `01.04.2026` | Расписание на конкретную дату |
| `преподаватель Павлова` | Расписание преподавателя |
| `сменить группу` | Изменить запомненную группу |
| `помощь` | Список всех команд |

Кнопки «Сегодня», «Завтра», «Моя группа», «Сменить группу», «Помощь» доступны в VK-клавиатуре.

## База данных

SQLite (`data/database.db`), три таблицы:

- **users** — пользователи бота: `vk_id`, имя, запомненная группа, состояние FSM-диалога
- **messages** — история сообщений: входящий текст, ответ бота, timestamp
- **schedule** — занятия: группа, дата, время начала/конца, предмет, тип занятия (лекция/практика/лаб.), преподаватель, аудитория, подгруппа

## Как работает интеграция с YandexGPT

Бот использует две стратегии в зависимости от запроса:

**Точечные запросы** (группа + дата, преподаватель, конкретная дата) — данные берутся из SQLite и передаются прямым контекстом в YandexGPT через Foundation Models API (`llm.api.cloud.yandex.net`). Vector Store в этом случае не используется.

**Свободные запросы** (без чёткой группы и даты) — используется Responses API с `file_search` по Yandex AI Studio Vector Store. Расписание каждой группы хранится в Vector Store как отдельный текстовый файл.

После каждого обновления расписания `yandex_kb_sync.py` автоматически заменяет файл группы в Vector Store: удаляет старый и загружает новый.

## Технологии

- **Python 3.14**
- **vk_api** — работа с VK Bots LongPoll API
- **SQLAlchemy 2.x** — ORM, SQLite
- **BeautifulSoup4 + requests** — парсинг сайта ИРНИТУ
- **openai** — OpenAI-совместимый клиент для Yandex AI Studio
- **schedule** — планировщик фонового обновления
- **python-dotenv** — конфигурация через `.env`
