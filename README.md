#  Schedule Assistant — Бот расписания ИРНИТУ

VK-бот для получения расписания занятий Иркутского национального исследовательского технического университета (ИРНИТУ).

##  Возможности

-  Расписание любой группы института (40+ групп)
-  Расписание на конкретную дату
-  Расписание преподавателя
-  Быстрые кнопки «Сегодня» и «Завтра»
-  Бот запоминает группу пользователя
-  Автоматическое обновление расписания каждые 24 часа

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone <url>
cd scheldule_assistant
```

### 2. Создать виртуальное окружение
```bash
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # Linux/Mac
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Создать файл `.env`
```env
VK_TOKEN=ваш_токен_сообщества_вк
GROUP_ID=id_вашего_сообщества_вк
```

### 5. Запустить
```bash
python app.py
```

При первом запуске бот автоматически скачает расписание всех групп (~5-10 минут в фоне). Бот начнёт отвечать сразу.

##  Команды бота

| Команда | Описание |
|---|---|
| Название группы (АСУб-25-1) | Показать расписание группы |
| `сегодня` | Расписание на сегодня |
| `завтра` | Расписание на завтра |
| `01.04.2026` | Расписание на конкретную дату |
| `преподаватель Павлова` | Расписание преподавателя |
| `помощь` | Список команд |

##  База данных

SQLite (`data/database.db`), три таблицы:

- **users** — пользователи бота (vk_id, имя, запомненная группа, состояние диалога)
- **messages** — история сообщений (для анализа и отладки)
- **schedule** — расписание занятий (группа, дата, время, предмет, тип, преподаватель, аудитория)

##  Технологии

- **Python 3.14**
- **vk_api** — работа с VK API
- **SQLAlchemy** — ORM для базы данных
- **BeautifulSoup4** — парсинг HTML
- **requests** — HTTP запросы
- **python-dotenv** — конфигурация через .env
